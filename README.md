# HPC_gauss_omp_mpi

Практическая работа по предмету "Суперкомпьютеры и параллельная обработка данных"
ВМК МГУ, 2025 год

Структура проекта

./opt_results - результаты работы скрипта, компилирующего программы для сравнения оптимизаций.
./scaling_results - результаты работы скрипта, компилирующего программы для исследования масштабируемости параллельных версий программ.

./code - директория с реализованными программами (язык С)
gauss-omp-for.c - оптимизированная версия программы (for)
gauss-omp-task.c - оптимизированная версия программы (task)
gauss-mpi.c - оптимизированная версия программы (mpi)
gauss-seq.c - оригинальная программа, но, тем не менее, с изменением. Теперь можно считать аргументы из командной строки. Она была модифицирована. Тем не менее, алгоритм остался тем же самым.

./python-scripts - директория с питон-скриптами, в которой собраны программы, выполняющие визуализации и сбор данных
collect_data.py - скрипт, собирающий данные
plot_results.py - скрипт для вывода всех графиков, в том числе тех, которые я не вставил пока что в отчет.
.scripts/ - скрипты для компиляции программ и их запуска с различным набором параметров.

./text - тексты отчета, формат word

Замечания

Тестирование было преимущественно с использованием -О2 оптимизатора.  
Запуск каждой версии производился 3 раза с последующим усреднением результата
