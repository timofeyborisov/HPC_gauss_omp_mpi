full_data - все графики и собранные данные
raw_data - данные, полученные в результате запуска script.sh
collect_data.py - скрипт, собирающий данные
gauss-omp.c - оптимизированная версия программы
gauss-unoptimized.c - оригинальная программа, но (!) всё же с изменением. Теперь можно считать аргументы из командной строки.
plot_results.py - скрипт для вывода всех графиков, в том числе тех, которые я не вставил пока что в отчет.
script.sh - скрипт для компиляции программ и их запуска с различным набором параметров.
В финальной версии на полюс отправил 280 заданий, перебрав все количества нитей, размеры массивов, а также запустив каждую программу 5 раз.
Тестирование было только с использованием -О2 оптимизатора.
